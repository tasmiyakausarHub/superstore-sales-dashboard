# 📊 Superstore Sales and Profitability Dashboard

## 📌 Project Overview
This project analyzes sales and profit data using Power BI. It provides insights into sales trends, profitability, and regional performance.

---

## 📁 Dataset
- Source: Superstore Dataset
- Columns include:
  - Product Name
  - Category
  - Sales
  - Profit
  - Quantity
  - Discount
  - Region
  - Order Date

---

## 📊 Dashboard Features
